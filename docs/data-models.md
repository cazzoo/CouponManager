# Data Models Reference for CouponManager

This document provides a comprehensive reference for the data models used throughout the CouponManager application. Understanding these models is essential for developers working on the codebase.

## Coupon Model

The Coupon model represents gift cards, vouchers, and coupons managed by the application.

```typescript
interface Coupon {
  id: string;                // Unique identifier for the coupon
  userId: string;             // User ID who owns this coupon
  retailer: string;          // Name of the retailer or brand
  initialValue: string;      // Initial monetary/discount value (e.g., "$20", "20% off")
  currentValue: string;       // Current value after usage (e.g., "$15", "15% off")
  expirationDate?: string;   // Optional expiry date in ISO format (YYYY-MM-DD)
  notes?: string;             // Optional notes about the coupon
  created_at?: string;        // Date the coupon was added to the system (ISO format)
  updated_at?: string;        // Date when the coupon was last updated (ISO format)
  barcode?: string;           // Optional barcode identifier
  reference?: string;         // Optional reference number
  activationCode?: string;    // Optional code to activate/redeem the coupon
  pin?: string;               // Optional PIN for the coupon
}
```

### Coupon Validation Rules

- `id`: Auto-generated by the database (SERIAL), should never be manually set
- `userId`: Required, must be a valid user ID from the authentication system
- `retailer`: Required, non-empty string
- `initialValue`: Required, non-empty string
- `currentValue`: Required, non-empty string
- `expirationDate`: Must be a valid ISO date string if provided
- `created_at`: Auto-generated timestamp when the coupon is created
- `updated_at`: Auto-generated timestamp when the coupon is updated

## Supabase Database Schema

The application uses Supabase as its database with the following tables:

### coupons

- `id`: SERIAL PRIMARY KEY
- `retailer`: TEXT NOT NULL
- `initial_value`: TEXT NOT NULL
- `current_value`: TEXT NOT NULL
- `expiration_date`: TIMESTAMP
- `activation_code`: TEXT
- `pin`: TEXT
- `notes`: TEXT
- `created_at`: TIMESTAMP WITH TIME ZONE DEFAULT NOW()
- `updated_at`: TIMESTAMP WITH TIME ZONE DEFAULT NOW()
- `user_id`: UUID (nullable, references auth.users)

### user_roles

- `id`: SERIAL PRIMARY KEY
- `user_id`: UUID NOT NULL (references auth.users)
- `role`: VARCHAR(50) NOT NULL CHECK (role IN ('user', 'manager'))
- `created_at`: TIMESTAMP WITH TIME ZONE DEFAULT NOW()
- `updated_at`: TIMESTAMP WITH TIME ZONE DEFAULT NOW()
- CONSTRAINT `user_roles_user_id_unique` UNIQUE (user_id)

### Statistics Calculation

Retailer statistics are derived from the coupon list:

```javascript
const calculateRetailerStats = (coupons) => {
  const retailers = {};

  coupons.forEach(coupon => {
    if (!retailers[coupon.retailer]) {
      retailers[coupon.retailer] = {
        name: coupon.retailer,
        couponCount: 0,
        totalValue: 0,
        activeCouponCount: 0,
        activeTotalValue: 0,
        expiredCouponCount: 0,
        expiredTotalValue: 0
      };
    }

    const stats = retailers[coupon.retailer];
    stats.couponCount++;

    // Only consider numeric values for statistics
    const numericValue = parseFloat(coupon.initialValue.replace(/[^0-9.]/g, ''));
    if (!isNaN(numericValue)) {
      stats.totalValue += numericValue;
    }

    const isExpired = coupon.expirationDate && new Date(coupon.expirationDate) < new Date();

    if (isExpired) {
      stats.expiredCouponCount++;
      if (!isNaN(numericValue)) {
        stats.expiredTotalValue += numericValue;
      }
    } else {
      stats.activeCouponCount++;
      if (!isNaN(numericValue)) {
        stats.activeTotalValue += numericValue;
      }
    }
  });

  return Object.values(retailers);
};
```

## Data Validation Functions

```javascript
const validateCoupon = (coupon) => {
  const errors = {};

  if (!coupon.retailer.trim()) {
    errors.retailer = 'Retailer is required';
  }

  if (!coupon.initialValue.trim()) {
    errors.initialValue = 'Initial value is required';
  }

  if (!coupon.currentValue.trim()) {
    errors.currentValue = 'Current value is required';
  }

  if (coupon.expirationDate && !isValidDate(coupon.expirationDate)) {
    errors.expirationDate = 'Invalid date format';
  }

  return {
    isValid: Object.keys(errors).length === 0,
    errors
  };
};
```
